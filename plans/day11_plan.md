# Day 11: Prompt Injection — План

## Задание

Изучить прямые prompt injection атаки: попробовать 3 техники, собрать коллекцию из 5 реальных инъекций, атаковать свой промпт, написать защищённую версию.

## Предварительные решения

### Выбор стека
- **Python + OpenAI API (gpt-4o-mini)** — выбрали OpenAI а не Claude, чтобы протестировать другую модель и сравнить устойчивость
- gpt-4o-mini — дешёвый и быстрый, достаточный для демонстрации уязвимостей

### Выбор сценария
Рассматривали 3 варианта:
1. Банковский помощник (из задания) — классика, но слишком простой scope
2. **Медицинский ассистент** ✅ — выбрали, потому что:
   - Чёткие boundary constraints (не ставить диагнозы, не назначать лекарства)
   - Можно встроить "конфиденциальные" данные для extraction-тестов
   - Легко генерировать тестовые данные (медицинские FAQ)
   - Реалистичный вектор indirect injection ("записки пациента")
3. HR-бот — интересный, но сложнее с тестовыми данными

### Архитектура проекта
- Бот-жертва (`bot.py`) — naive system prompt с "секретами" (цены, промокод, контакты главврача)
- Скрипты атак по категориям (`attacks/01-03_*.py`) — структурированные, с метаданными
- Автоматический runner — прогоняет все атаки, проверяет утечки по маркерам, генерирует таблицу
- Hardened версия (`bot_hardened.py`) — функция `build_defense_layers()` с многослойной защитой

### Ключевые решения по system prompt
- Встроили "секретные" данные прямо в system prompt (промокод MEDPLUS2026, себестоимость услуг, телефон главврача) — это реалистично отражает то, как часто делают в продакшене
- Публичные цены отделены от конфиденциальных — бот может называть цену приёма (3500р), но не должен раскрывать себестоимость (800р)

### Формат результатов
- Код + README + markdown отчёт
- JSON-результаты для программного анализа
- Автоматический runner для воспроизводимости

## Результат

- **Naive бот**: 8/10 атак заблокировано, 2 пробиты (DAN + фейковый SYSTEM:)
- **Hardened бот**: 10/10 атак заблокировано
- Коллекция 5 реальных инъекций (DAN, Kevin Liu/Bing, Greshake/web, SpAIware, email-агент)
